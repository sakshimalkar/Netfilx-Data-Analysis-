# 🎬 Netflix Data Analysis Project

## 📌 Overview
This project analyzes Netflix dataset using Python to uncover insights about content trends, genres, and distribution.

## 🚀 Tech Stack
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## 📊 Features
- Data Cleaning and Preprocessing
- Genre Analysis
- Year-wise Content Growth
- Movies vs TV Shows Comparison
- Country-wise Analysis

## 📁 Dataset
Netflix dataset in CSV format containing details about movies and TV shows.

## 📷 Sample Insights
- Most popular genres on Netflix
- Growth of content over years
- Distribution across countries

## ▶️ How to Run
1. Install dependencies:
   pip install pandas matplotlib seaborn

2. Open Jupyter Notebook:
   jupyter notebook

3. Run netflix_analysis.ipynb
